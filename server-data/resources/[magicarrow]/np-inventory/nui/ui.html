Üzerinde Kadar Var.

About:

Itemlerin Hepsini birden almak istiyorsanız miktar kısmını boş bırakın (Miktara bakın)
Rakamları veya fare tekerleğini kullanarak taşımak istediğiniz miktarı ayarlayabilirsiniz. Bunu yapabilirsiniz sürüklerken veya önceden.
To USE bir öğeyi orta alana sürükleyin.
AMOUNT alanı kapatıldığında sıfırlanmayacak, bu yüzden onu temizlemeye alışın!

Keybinds (defaults):

  • Shift + Click: Yığının yarısını karşı envantere taşır.
  • Ctrl + Click: AMOUNT alanına girilen miktarı karşı envantere taşır.
  • Left Click: Normal Sürükleme (ayarlarda Sağ Tıklama gibi olacak şekilde yapılandırılabilir)
  • Right Click: Sürüklemek için Tutmak Yok
  • Middle Click: Öğeyi kullanır
  • Scroll Wheel: AMOUNT üzerine gelindiğinde değer artacak/azalacaktır
  • Numbers 0 - 9: Yazılan numarayı girecek.
  • Any other key: AMOUNT değerini tüm yığını boşaltmak/taşımak için ayarlayacaktır.
  • Bir öğenin üzerine gelirken Ctrl + C tuşlarına basmak, açıklamayı kopyalayacaktır. kanıt numaraları.

Shops:

TUTAR alanına satın almak istediğiniz tutarı girin ve ardından mağaza öğesini envanter.

Crafting:

Make sure you know the requirements beforehand for any crafting menu that takes a long time to open.
When crafting, you have to put in the exact amount you can craft.

Item Durability:

Almost all items will decay over time (including weapons) The bar under each item will slowly decrease in size as it decays more and more, then eventually fade to a reddish color. Once an item is fully destroyed, it will have a solid red bar.
